What it does

Text extraction without plugins or cloud OCR.

Works over other apps

Capture text from screenshots, chat images, browser images, and image viewers with a region overlay.

Selectable overlay

Hover recognized words, drag-select text, use Ctrl + A, and copy with Ctrl + C.

Tray-first workflow

TextLayer stays in the notification area and is ready when you need OCR, with near-zero idle activity.

How it works

Three steps.

  1. Launch TextLayer. It starts as a tray app.
  2. Press Ctrl+Shift+O. Drag over the image region you want to OCR.
  3. Select and copy. Use the overlay like a text layer and close it with Esc.

Privacy

OCR stays local.

TextLayer uses local Windows OCR and local Tesseract OCR assets. Captured images are not sent to a server, and there is no telemetry or cloud fallback.
