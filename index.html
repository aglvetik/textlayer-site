Why TextLayer exists

Copy text where apps do not allow selection

Useful for screenshots, messengers, launchers, game UI, support dialogs, and images opened in other apps.

Keep recognition local

Captured regions are processed on your Windows machine instead of being sent to an online OCR service.

Use it as a desktop utility

TextLayer is designed as a small Windows tool that stays available when you need to work with visual text.

Where it helps

  • Screenshots with text
  • Discord or Telegram images
  • Error messages and logs shown in screenshots
  • Game UI or server information screenshots
  • Scanned fragments and document previews
  • Text inside browser images

TextLayer is for moments when text is visible, but not selectable. Instead of retyping it manually, you can extract it locally and paste it where you need.

Technical note

Default capture hotkey: Ctrl+Shift+O

Private by default

TextLayer does not upload captured images to a server. Recognition runs locally on your computer.

  • No cloud OCR
  • No telemetry
  • No browser extension
  • No continuous screen-scanning loop
