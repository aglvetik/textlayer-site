Features

Built for quick local text capture.

Capture text from images and apps

Select any region on screen and extract text from screenshots, dialogs, browsers, or image viewers.

Select and copy recognized text

The result appears as a text overlay so you can highlight, select, and copy what you need.

Runs with local OCR only

Recognition uses Windows OCR and local Tesseract assets. Nothing is sent to a remote service.

Tray app with a simple hotkey flow

Keep TextLayer in the notification area and start capture whenever you need it.

How it works

Four simple steps

  1. Launch TextLayer

    The app starts in the tray and stays ready in the background.

  2. Press Ctrl+Shift+O

    Start a capture without switching apps or opening a separate window.

  3. Select a region

    Drag over the part of the screen that contains the text you want to read.

  4. Copy recognized text

    Select the OCR result from the overlay and copy it like normal text.

Privacy

Your OCR stays on your device.

TextLayer is designed to work fully locally. Captured images and recognized text are processed on the machine and not uploaded anywhere.

  • OCR runs on-device
  • Captured regions are not uploaded
  • No telemetry or analytics
  • No cloud fallback
