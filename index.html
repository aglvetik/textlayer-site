Region capture

Recognize only the part of the screen you need.

Selectable overlay

TextLayer places recognized text over the captured area so you can copy it normally.

Local OCR

OCR runs on your Windows machine using local OCR engines.

English and Russian

The app UI supports English and Russian.

Windows desktop app

Runs as a normal Windows utility and can stay available from the tray.

No browser extension

Works with images shown in Discord, Telegram, browsers, image viewers, and other apps.

Private by design

TextLayer does not need cloud OCR. Captured screen regions are processed locally on your computer.

  • No image upload
  • No cloud OCR
  • No telemetry
  • No continuous screen scanning

  • Region capture currently works per selected screen area.
